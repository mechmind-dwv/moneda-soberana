# Moneda Social Tonkerizada 🚛💰

Sistema de crédito mutuo tokenizado con gobernanza ciudadana y demurrage automático.

## Estructura
- `contracts/`: Smart contracts en Solidity (crédito mutuo, identidad ZK, DAO)
- `frontend/`: Monedero móvil (React Native) y portal de gobernanza
- `governance/`: Configuración de despliegues y ABI de la DAO
- `identity/`: Circuitos ZK para verificación de persona única
- `docs/`: Documentación técnica y comunitaria
